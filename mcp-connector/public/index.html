Outlook

Microsoft Graph API — OAuth 2.0
Not configured
📋 How to find your Outlook credentials ▼
  1. Go to portal.azure.com and sign in.
  2. Search for Microsoft Entra ID → App registrations → New registration.
  3. Name your app, set Redirect URI to http://localhost:4000/auth/outlook/callback (Web platform).
  4. After creation, copy the Application (Client) ID and Directory (Tenant) ID from the Overview page.
  5. Go to Certificates & secrets → New client secret. Copy the secret Value (not the Secret ID).
  6. Under API permissions → Add a permission → Microsoft Graph → Delegated, add:
    User.Read  Mail.Send  Calendars.Read  offline_access
  7. Click Grant admin consent for your organization (same step as Teams — prevents the "justification required" prompt).
  8. Save your credentials below, then click Connect with Microsoft to authorize.
⚠️ Always copy the secret Value, not the Secret ID. The wrong field causes error AADSTS7000215.
Azure Portal → App registrations → Overview
Certificates & secrets → Value column (not the ID column)
Azure Portal → Microsoft Entra ID → Overview

Microsoft Teams

Microsoft Graph API — OAuth 2.0
Not configured
📋 How to find your Teams credentials ▼
  1. Go to portal.azure.com and sign in.
  2. Search for Microsoft Entra ID → App registrations → New registration.
  3. Name your app, set Redirect URI to http://localhost:4000/auth/teams/callback (Web platform).
  4. After creation, copy the Application (Client) ID and Directory (Tenant) ID from the Overview page.
  5. Go to Certificates & secrets → New client secret. Copy the secret Value (not the Secret ID).
  6. Under API permissions → Add a permission → Microsoft Graph → Delegated, add:
    User.Read  Calendars.Read  OnlineMeetings.Read  OnlineMeetingRecording.Read.All  OnlineMeetingTranscript.Read.All  offline_access
  7. Click Grant admin consent for your organization (required for Recording/Transcript scopes).
  8. Save your credentials below, then click Connect with Microsoft to authorize.
⚠️ Recording and Transcript scopes require admin consent. If your org restricts this, ask your Microsoft 365 admin to approve the app.
⚠️ For transcripts to appear, transcription must be enabled before the meeting starts.
In Teams Admin Center → Meetings → Meeting policies → enable Transcription. Then in the meeting click More → Start transcription.
Azure Portal → App registrations → Overview
Certificates & secrets → Value column (not the ID column)
Azure Portal → Microsoft Entra ID → Overview

3CX SMS

Telnyx SMS Gateway
Not configured
📋 How to find your Telnyx credentials ▼
  1. Log in to telnyx.com and go to API Keys in the left menu.
  2. Click Create API Key, give it a name, and copy the key immediately (it won't be shown again).
  3. Go to Numbers → find your SMS-enabled phone number. Copy it in E.164 format (e.g. +15551234567).
  4. Paste both values below and click Save, then Test Connection.
ℹ️ This is the same Telnyx account that 3CX uses as its SMS provider — no extra setup needed.
telnyx.com → API Keys
E.164 format — telnyx.com → Numbers
